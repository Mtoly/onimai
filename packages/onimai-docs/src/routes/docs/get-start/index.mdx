# Getting Start

## Install

```shell
yarn add onimai @onimai/qwik
```

Then modify your `tailwind.config.js`:

```js
module.exports = {
  // Add `./node_modules/@onimai/qwik/**/*.{cjs,mjs}` in content
  content: [
    // ...
    "./node_modules/@onimai/qwik/**/*.{cjs,mjs}"
  ]
  // Add `onimai` in plugins
  plugins: [
    // ...
    require("onimai")
  ],
};
```
