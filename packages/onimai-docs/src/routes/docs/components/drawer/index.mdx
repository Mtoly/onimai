## Usage

```jsx
// id should be globally unique, anything :checked can trigger drawer to open
<input type="checkbox" hidden id="drawer-x" />
<div class="drawer">
  <label class="drawer-background" for="drawer-x" />
  <div class="drawer-content p-4">
    <h2 class="text-2xl font-bold">Hello world</h2>
    <p>Onimai is the best anime I've ever seen!!!</p>
  </div>
</div>
```
