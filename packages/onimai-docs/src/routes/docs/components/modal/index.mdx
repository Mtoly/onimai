## Usage

```jsx
// id should be globally unique, anything `:checked` can trigger modal to open
<label class="btn" for="modal-x" tabIndex={0}>Example</label>
<input type="checkbox" hidden id="modal-x">
<div class="modal">
  <div class="modal-background" />
  <div class="modal-content">
    <h2 class="text-2xl font-bold">Create new waifu</h2>
    <p class="opacity-60">Will you marry Mahiro?</p>
    <p><label class="btn" for="modal-x" tabIndex={0}>Yes</label></p>
  </div>
</div>
```

## Options

| Field      | Type       | Description                                                                      |
| ---------- | ---------- | -------------------------------------------------------------------------------- |
| `id`       | `string`   | A globally unique id for this modal, you can use `<labal for="id" />` to open it |
| `closable` | `boolean?` | Set `false` to remove top-right close button and background label element        |
