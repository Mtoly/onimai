# Modals

## Usage

```jsx
// modal opener
<label class="btn" for="global-unique-id" tabIndex={0}>
  Modal Open Button
</label>
// modal body
<Modal id="global-unique-id">
  {/* Default contents */}
  <h2 class="text-2xl font-bold">Create new waifu</h2>
  <p class="opacity-60">Are you sure to mark Mahiro as your new waifu?</p>
  {/* Actions in slot action */}
  <button class="btn" q:slot="action">
    Yes
  </button>
  <label class="btn" for="global-unique-id" tabIndex={0} q:slot="action">
    No
  </label>
</Modal>
```

## Options

| Field      | Type       | Description                                                                      |
| ---------- | ---------- | -------------------------------------------------------------------------------- |
| `id`       | `string`   | A globally unique id for this modal, you can use `<labal for="id" />` to open it |
| `closable` | `boolean?` | Set `false` to remove top-right close button and background label element        |
